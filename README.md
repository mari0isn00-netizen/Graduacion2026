# 📺 SEÑAL G - Fiesta de Graduación Interactiva

Un programa de TV retro pero innovador para tu fiesta de graduación.

## 🚀 Instrucciones de Uso

1. Clona el repositorio
2. `npm install`
3. `npm start`
4. Conecta la TV a `/tv`
5. Admin accede a `/admin`
6. Jugadores entran por `/player` escaneando QR

## 🎮 Juegos Incluidos
- 🎵 Música y Stop
- 💣 La Bomba

- 💬 ¿Quién lo Dijo?
- 🫙 Esconder Objeto
