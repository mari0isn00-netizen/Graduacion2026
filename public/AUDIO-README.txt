Para que la bienvenida tenga música, necesitas:

  1. Descargar "DTMF" de Harry Styles
     (disponible en Spotify, YouTube, Apple Music, etc.)

  2. Convertirla a MP3 si es necesario
     Puedes usar: https://ytmp3.nu o cualquier conversor

  3. Renombrar el archivo exactamente como:
        dtmf.mp3

  4. Colocarlo en esta misma carpeta (/public/)
     junto a este archivo README

  5. Listo. El sistema la detectará automáticamente.
